# 에너지 텐션 그래프에 대한 코멘트
- 현재 UX 반응이 직관적이지 않음.
- 에너지 그래프의 임의의 좌표에서 우클릭 시 [여기에 에너지 단계 추가 / 이 에너지 단계를 제거]가 나오도록 하기.
- 여기에 에너지 단계 추가 : 현재의 구간 분리선 추가와 동일. 추가 시 우클릭 시점의 x좌표에 구간 분리선을 생성하고 새로 생긴 구간의 에너지 값은 앞 뒤 값의 평균값.
- 이 에너지 단계를 제거 : 우클릭한 좌표에서 가장 가까운 에너지 포인트(보라색 동그라미)의 오른쪽에 있는 구간 분리선(빨강 마름모)을 제거. 가장 마지막(오른쪽)에 있는 영역을 제거 시 마지막 구간을 지우고 마지막 직전 영역이 마지막 시간까지 대신하게 됨.

# 기타사항
- 세부 설정의 안내문구는 박스 영역을 만들지 말고 텍스트만 넣을 것. 폰트 스타일은 가장 아래 있는 팬메이드 명시 문구와 동일한 스타일로 작성하면 됨.
