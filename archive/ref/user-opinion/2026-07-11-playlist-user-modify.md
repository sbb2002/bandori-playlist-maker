# 추가 기능 제안

- 현재 : 플레이리스트가 한번 생성되고 나면 일부 수정하기 어려움. 따라서 아래 기능을 추가하고자 함.

- 새 기능
    * 순서 이동
        - 생성된 곡을 위아래로 드래그하여 순서를 이동시킬 수 있음. 
        - (optional) 이동 후 에너지 텐션 그래프 업데이트

    * 곡 제거
        - 생성된 곡을 제거함.
        - (optional) Ctrl+Z를 통해 되돌리기를 할 수 있음.
        - (optional) 이동 후 에너지 텐션 그래프 업데이트

    * 곡 추가
        - 사용자가 원하는 자리에 임의의 곡을 추가할 수 있음. 기존의 곡을 선택하는 방법이 필요. bandori-song-sorter의 밴드 셀렉터와 곡 리스트를 미니버전으로 구현하여 활용.
        - (optional) 이동 후 에너지 텐션 그래프 업데이트
    
    * 곡 영역 호버링 이벤트
        -곡 위에 마우스를 올려놓으면 해당 영역이 약간 밝게 강조 및  영역의 우측에 위아래 "이동가능 표시" 버튼, "-" 버튼, "+" 버튼 등장
        - 이동가능 표시 버튼을 드래그하면 순서 이동.
        - "-" 버튼을 클릭하면 해당 곡을 지움.
        - "+" 버튼을 클릭하면 다음 순서에 새 곡 추가.
