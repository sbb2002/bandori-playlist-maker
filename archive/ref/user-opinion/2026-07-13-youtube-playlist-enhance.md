# YouTube Playlist OAuth Implementation Plan

## 0. 목적

현재 플레이리스트 생성 후 'Youtube 재생목록 (만들기)' 버튼을 클릭하여 'url 복사'와 '유튜브에서 듣기'를 지원한다.
이를 좀 더 발전시켜 url 복사는 지금처럼 남겨놓되, '유튜브에서 듣기' 기능을 '내 재생목록에 넣기' 기능으로 전환하고자 한다.
'내 재생목록에 넣기' 기능은 사용자 각자 로그인된 계정으로 재생목록을 생성하는 것을 의미한다.


## 1. 지금 듣기(현재 기능)

- 웹 플레이어(IFrame)에서 즉시 재생
- 기존 동작 유지


## 2. URL 복사(현재 기능)

- 커뮤니티 공유용
- 임시 플레이리스트 생성
- 기존 동작 유지


## 3. 내 YouTube 재생목록 만들기(기존 '유튜브에서 듣기' 버튼의 대체 기능)

- 기존 기능을 대체하는 신규 기능
- 동작 순서
    ```text
    사용자 클릭 → Google 로그인(이미 로그인된 경우 생략) → 새 Playlist 생성 → 현재 Playlist의 video_id 추가 → 생성된 Playlist 열기
    ```
- 중요한 점은 **watch_videos URL을 다시 사용하는 것이 아니라** 현재 생성되어 있는 플레이리스트의 `video_id` 목록을 그대로 이용하여 YouTube Playlist를 생성하는 것이다.
- Google 로그인 또는 권한 승인에 실패(취소 포함)하면 기존 플레이리스트는 유지하고, 재생목록 생성은 수행하지 않는다.


## 4. 구현 원칙

- 기존 watch_videos 기능은 그대로 유지한다.
- 공유 기능과 저장 기능을 분리한다.
- 저장 기능만 OAuth를 사용한다.
- 저장되는 Playlist는 사용자 자신의 계정에 생성된다.
- 개발자 계정에는 생성되지 않는다.
- 현재 메모리에 존재하는 video_id 목록을 그대로 이용하여 Playlist를 생성한다.
- watch_videos URL을 다시 파싱하지 않는다.
