# 하단 플레이바

- 새로운 피쳐 : 화면의 하단에 항상 떠있는 플레이 바를 만들기.

- 플레이리스트 생성 전에는 등장하지 않다가 플레이리스트 생성 시 아래에서 플레이바가 위로 올라옴.

- '곡 이름, 밴드명, 현재곡의 현재시간/현재곡의 종료시간, n(현재 재생곡 순서번호)/N(마지막 순서번호)'를 표시해야함.

- '이전곡, 재생(일시정지), 다음곡, 반복' 버튼을 넣어야 함.

- '재생 프로그레스 바'는 1초마다 업데이트되어야 하고, 사용자가 재생 위치를 조절할 수 있어야 함. 만약 동기화가 어렵다면 프로그레스 바는 생략하는 대신 곡 이름 영역 및 밴드명 영역 클릭 시 유튜브프레임으로 스크롤 이동하는 것으로 대체.

- pc, 모바일 모두 동작해야하므로 화면 비율에 주의할 것.

- UI의 초안 
    ```
    [곡 제목] 0:00 ━━━━○────────── 3:30 ⇆ ◁ ❚❚ ▷ ↻
    ```
