# GEMS-9 설문 응답 가이드 (피설문자용)

대상 파일: `out/gems9_pilot_candidates.csv`

## 진행 순서
- 한 행 = 한 곡. 위에서부터 순서대로 진행.
- `url`을 열어 `excerpt_start_sec`~`excerpt_end_sec` 구간만 듣기.
- 구간 반복 청취 금지 — 한 번 듣고 바로 채점.
- 다 듣고 나서 9개 항목에 1~5점으로 채점.
- 가사 내용이 아니라 **곡 전체 느낌**(멜로디·연주·분위기)으로 채점.
- 정답 없음 — 첫 느낌대로.

## 점수 기준 (모든 항목 공통)
- 1점: 전혀 못 느낌
- 2점: 약간 느낌
- 3점: 보통으로 느낌
- 4점: 꽤 강하게 느낌
- 5점: 매우 강하게 느낌

## 9개 항목 뜻
- `wonder` (경이): 우와 신기하다, 놀랍다는 느낌
- `transcendence` (초월): 성스럽거나 신비로운, 뭔가 큰 것과 이어진 느낌
- `tenderness` (다정함): 부드럽고 사랑스러운 느낌
- `nostalgia` (향수): 옛 생각나는, 그리운 느낌
- `peacefulness` (평온함): 마음이 편안하고 고요해지는 느낌
- `power` (웅장함): 힘이 넘치고 강한, 이겨내는 듯한 느낌
- `joyful_activation` (활기): 신나고 즐거운, 들뜨는 느낌
- `tension` (긴장): 불안하고 초조한 느낌
- `sadness` (슬픔): 슬프고 처연한 느낌

## 참고
- 한 곡에서 여러 항목이 동시에 높게 나와도 됨(예: `power`와 `sadness` 둘 다 4점).
- `rater_note`에 짧은 인상 메모 남기면 좋음(선택).
- 한 번에 몰아서 채점 — 다른 날 나눠서 하면 기준이 흔들릴 수 있음.
