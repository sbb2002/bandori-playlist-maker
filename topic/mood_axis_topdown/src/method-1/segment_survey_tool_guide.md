# 설문 응답 툴 사용설명서 (segment_survey_tool.html)

## 실행 방법
- Tool 1(구간 설정 툴)과 동일하게 로컬 서버로 실행 권장:
  ```
  cd topic/mood_axis_topdown/src/method-1
  python -m http.server 8000
  ```
  `http://localhost:8000/segment_survey_tool.html` 접속.

## 구간이 아직 안 정해진 곡은 어떻게 되나
- 구간 미확정 곡은 자동으로 **인트로 0:00~0:30**(폴백)로 재생됨.
- 화면 상단에 **"구간 임시값(미확정)"** 배지가 뜨는 곡은 아직 정식 구간이 아님 — 참고만 할 것.
- Tool 1에서 구간 설정을 끝내고 CSV를 갱신한 뒤,
  ```
  python build_gems9_survey_data.py
  ```
  한 번만 실행하면 이 툴의 곡 데이터가 정식 구간으로 자동 교체됨(재작성 불필요).

## 사용 순서
1. **"▶ 재생"** 클릭 → 설정된 구간만 자동 재생, 끝나면 자동 정지·처음으로 복귀.
2. 듣는 동안/듣고 나서 9개 항목에 1~5점 클릭(다시 클릭하면 점수 변경 가능).
3. 필요하면 **"■ 정지"**로 즉시 중단(처음 위치로 복귀).
4. 인상 메모는 선택 입력.
5. **"다음"/"이전"** 또는 **"곡 목록"** 메뉴로 이동.
6. 다 끝나면 **"CSV 내보내기"**.

## 채점 기준
- 1=전혀 못 느낌, 3=보통, 5=매우 강하게 느껴짐(화면 상단 범례 참고).
- 9개 항목 뜻: `notes/gems9_response_guide.md` 참고.

## 저장·내보내기
- 응답은 브라우저에 자동 저장(새로고침해도 유지, 같은 브라우저·기기 한정).
- 내보내기 파일: `gems9_survey_responses.csv`(로컬) / `.txt`(아티팩트, 확장자 변경 필요).
- 영상은 재생/정지 버튼으로만 제어되며, 영상 자체를 클릭해 탐색할 수 없음(의도된 동작).
