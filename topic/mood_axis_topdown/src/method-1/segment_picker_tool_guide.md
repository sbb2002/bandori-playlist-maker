# 구간 설정 툴 사용설명서 (segment_picker_tool.html)

## 실행 방법
- 아티팩트(claude.ai)로 열면 유튜브 iframe이 동작하지 않을 수 있음 — **로컬 서버로 실행 권장**.
- `file://`로 직접 열지 말 것 — 유튜브 IFrame API 통신이 막힐 수 있음.
- 실행:
  ```
  cd topic/mood_axis_topdown/src/method-1
  python -m http.server 8000
  ```
  이후 브라우저에서 `http://localhost:8000/segment_picker_tool.html` 접속.

## 화면 구성
- 상단: 진행 상황("N / 35 완료"), 곡 목록 버튼
- 중앙: 유튜브 영상(자유 재생·탐색 가능)
- 영상 아래: 재생구간 타임라인 바(클릭하면 그 위치로 이동)
- 그 아래: IN(시작점)·OUT(끝점) 입력칸 + 버튼

## 사용 순서
1. 영상 재생하며 원하는 구간 탐색.
2. 시작점에서 **"현재 위치로 설정"**(IN) 클릭 → 자동 입력.
3. 끝점에서 **"현재 위치로 설정"**(OUT) 클릭 → 자동 입력.
4. 인트로를 쓰고 싶으면 **"인트로 구간(0:00~0:30)"** 버튼으로 한 번에 설정 가능.
5. 구간 길이는 자동으로 판정됨(15~45초 권장 범위 벗어나면 경고색 표시).
6. 시작·끝 값은 입력칸에 직접 타이핑해서 수정도 가능(예: `1:05`).
7. **"다음"/"이전"** 또는 **"곡 목록"** 메뉴로 다른 곡 이동.
8. 모든 곡을 다 정했으면(또는 중간에라도) **"CSV 내보내기"** 클릭.

## 저장·내보내기
- 입력한 값은 브라우저에 자동 저장됨(새로고침해도 안 사라짐, 같은 브라우저·기기 기준).
- 내보내기 파일명은 `gems9_pilot_candidates_segments.csv`(로컬 실행 시) 또는
  `.txt`(claude.ai 아티팩트로 열었을 경우 — 확장자 제한 때문. 받은 뒤 `.csv`로 변경).
- 내보낸 파일로 기존 `out/gems9_pilot_candidates.csv`를 덮어쓰면 됨(단 GEMS 채점 칸은
  비어있는 상태로 나옴 — 그건 설문 툴(Tool 2)에서 채움).

## 참고
- 곡 목록의 점(●) 표시 = 그 곡의 시작·끝이 모두 설정됨.
- 다른 브라우저·다른 기기로 열면 저장된 값이 안 보임(로컬 저장이라 기기별로 따로 쌓임) —
  꼭 같은 환경에서 이어서 진행할 것.
