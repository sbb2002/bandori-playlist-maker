# 음원으로부터 가사 추출 및 벡터화 방법 제안

> **2026-07-17 업데이트**: 아래 초안에서 뽑은 감정 카테고리 4종(C1~C4)은 음향 confound와
> 카테고리 간 어휘 중복 문제로 재설계됨. 현재 방식은 `DESIGN.md` §1b, 실제 값은
> `src/method-1/config.py` 참조.

1. 음원에서 보컬을 분리(기존 demucs를 사용)

2. 분리된 보컬 음성으로부터 가사 추출
    - 모두 일어 및 영어(영문의 일어 음차도 섞여있을 수 있음.)임.
    - whisper 고려 중 (노래이기 때문에 음높이가 바뀌는 점을 감안해서 더 좋은 모델이 있다면 그것으로 스위칭해도 됨.)
    - 가사가 문장 형식이 아닌 의성어나 강조어만 있을 수 있음. 따라서 이에 대해 raw로 쓸지, 전처리하여 사용할지 방법론은 정해지지 않음.
    - 가사에서 chorus 부분에 해당하는 반복되는 가사가 있을 수 있음. 이에 대한 전처리는 고려 중.

3. (선택) 키워드 위주 산출 또는 문장 그대로 사용

4. 벡터화


# 테스트 방안

1. 아래 프롬프트에 대해 벡터DB에 서칭하여 곡 후보 TOP 5개 선택.

```
예시 1. [슬픔 / 우울] 계열
가장 흔히 입력되는 감정이자, 깊어질수록 가련함과 서글픈 미학이 살아나는 정서입니다.

- 1단계 (1차원적 감정): "우울하고 슬픈 노래 틀어줘."

- 2단계 (상황과 공간의 부여): "비 오는 창밖을 보며 혼자 쓸쓸해하는 분위기의 노래."

- 3단계 (구체적인 정서와 감각의 대조): "차가운 빗소리 사이로 가만히 마음을 달래주는, 살짝 서글프지만 잔잔한 피아노 선율 같은 분위기."

- 4단계 (세련되고 세심한 감정 맥락 - 추천 최종형): "짙은 밤하늘 아래 홀로 남겨진 듯한 고독감이 밀려오지만, 애써 담담하게 슬픔을 받아들이며 조용히 내면을 위로하는 애절하고 서정적인 정서."


예시 2. [가련함 / 나아감] 계열 (서브컬처/감성 록 특화)
질주감(에너지)과 연약함(가사/코드)이 대비되는, 개발 중이신 앱의 핵심 타겟 정서 중 하나입니다.

- 1단계 (1차원적 감정): "희망찬데 슬픈 노래."

- 2단계 (동사와 행동의 추가): "슬프지만 힘을 내서 앞으로 달려 나가는 느낌."

- 3단계 (상반된 감정의 시적 대조): "위태롭고 상처투성이인 발걸음이지만, 끝내 저 멀리 반짝이는 빛을 향해 손을 뻗는 듯한 정서."

- 4단계 (세련되고 세심한 감정 맥락 - 추천 최종형): "금방이라도 부서질 것처럼 연약하고 서글픈 보컬의 목소리 뒤로, 세차게 몰아치는 드럼과 기타 사운드가 질주하며 불안 속에서도 끝내 딛고 일어나 나아가고자 하는 아련하고 가련한 의지."


예시 3. [힙함 / 세련됨 / 시티팝] 계열
드럼의 싱코페이션(엇박)과 화려한 텐션 코드가 연상되어야 하는 정서입니다.

- 1단계 (1차원적 감정): "힙하고 세련된 노래."

- 2단계 (도시적 감각의 부여): "드라이브하면서 듣기 좋은 감각적이고 트렌디한 사운드."

- 3단계 (사운드 질감과 무드의 묘사): "차가운 도시의 밤거리, 화려한 네온사인 아래를 미끄러지듯 달리는 여유롭고 몽환적인 그루브."

- 4단계 (세련되고 세심한 감정 맥락 - 추천 최종형): "지나치게 무겁지 않은 미디엄 템포 위에 재지(Jazzy)한 건반과 찰진 베이스 리듬이 얹혀, 도회적인 고독과 낭만이 교차하는 감각적이고 스타일리시한 무드."


예시 4. [밝음 / 아침 / 위로] 계열
아침의 따스함과 편안함, 과하지 않은 에너지를 조율하는 정서입니다.

- 1단계 (1차원적 감정): "아침에 듣기 좋은 밝은 노래."

- 2단계 (온도와 조도의 표현): "햇살을 받으며 차분하게 하루를 시작하는 편안한 느낌."

- 3단계 (청각적 자극과 가사의 뉘앙스): "커피 한 잔의 온기처럼, 따뜻하고 나지막한 목소리로 어깨를 다독여주는 무공해 같은 힐링."

- 4단계 (세련되고 세심한 감정 맥락 - 추천 최종형): "이른 아침의 맑은 공기와 부드러운 햇살이 스며들 듯, 어쿠스틱한 악기들이 만드는 포근한 공간감 속에서 불안을 걷어내고 긍정적인 온기를 불어넣는 나른하면서도 화사한 순간."
```

2. 연구자(개발자)가 직접 들어보고 설문을 실시함.
    - 1(별로 일치하지 않음)~5(아주 일치함) 점수로 평가함. 
    - 이는 csv 파일로 제공하여 사람이 직접 채워넣을 것.

3. 설문조사 결과를 보고 본 앱에 적용 가능한지, 그 효용성은 얼마나 될지 검토.
