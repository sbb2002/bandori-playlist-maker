개인정보처리방침

Bangdream Playlist Maker (Bandori Playlist Maker)

시행일자: 2026-07-13

본 서비스(이하 "서비스")는 'BanG Dream!'의 팬이 비상업적 목적으로 제작한 개인 프로젝트입니다. 서비스는 회원가입이나 로그인 없이 누구나 사용할 수 있으며, 아래는 서비스 이용 과정에서 어떤 정보가 어떻게 처리되는지를 설명합니다. 문의는 페이지 하단 연락처로 해주세요.

1. 서버에 저장되는 개인 정보

없습니다. 서비스는 별도의 회원 데이터베이스나 사용자 계정 시스템을 운영하지 않습니다. 입력하신 요청 문장(프롬프트)은 플레이리스트를 구성하는 데만 즉시 사용되고, 응답을 반환한 뒤 서버에 저장되지 않습니다.

요청 해석을 위해 프롬프트 텍스트가 LLM 제공사인 Groq로 전송됩니다(무드·에너지 해석 목적). Groq의 데이터 처리 방침은 Groq 자체 정책을 따릅니다.

2. 브라우저에만 저장되는 정보

생성한 플레이리스트를 "저장된 플레이리스트" 메뉴에서 다시 열어볼 수 있도록, 브라우저의 localStorage에 최대 50개까지 보관합니다. 이 데이터는 사용자의 기기에만 저장되며 서비스 서버로 전송되지 않습니다. 브라우저 데이터를 지우거나 메뉴에서 직접 삭제하면 함께 삭제됩니다.

3. Google 계정 연동 — "내 재생목록에 넣기" 기능

플레이리스트 생성 후 "내 재생목록에 넣기" 버튼을 누르면, 사용자 자신의 Google 계정으로 로그인해 사용자 본인의 YouTube 계정에 실제 재생목록(일부공개 / unlisted — 링크를 아는 사람만 열람 가능하며 검색에는 노출되지 않음)을 생성합니다.

  • 요청하는 권한(스코프): youtube.force-ssl — 오직 재생목록 생성과 곡 추가에만 사용하며, 그 외 채널 정보 열람·구독 관리 등 다른 어떤 동작도 수행하지 않습니다.
  • 토큰 처리: 로그인 후 발급되는 접근 토큰(access token)은 브라우저 메모리에서만 존재하며, 서비스 서버로 전송되거나 저장되지 않습니다. 페이지를 새로고침하거나 닫으면 즉시 사라집니다. YouTube 재생목록 생성 API 호출은 사용자의 브라우저가 Google 서버로 직접 수행합니다(서버 경유 없음).
  • 생성 위치: 재생목록은 반드시 사용자 본인의 계정에만 생성되며, 개발자 계정이나 다른 어떤 저장소에도 생성되지 않습니다.
  • 권한 철회: 언제든지 Google 계정 — 타사 앱 및 서비스 액세스 권한 페이지에서 이 앱의 접근 권한을 철회할 수 있습니다.

4. YouTube API 서비스 이용에 관한 고지

본 서비스의 "내 재생목록에 넣기" 기능은 YouTube API 서비스(YouTube API Services)를 이용합니다. 이 기능을 사용함으로써 사용자는 아래 약관 및 방침의 적용을 받습니다.

  • YouTube 서비스 약관: https://www.youtube.com/t/terms
  • Google 개인정보처리방침: https://policies.google.com/privacy

4-1. 데이터의 접근·이용·저장·공유

  • 접근(Access): 사용자가 "내 재생목록에 넣기"를 직접 누른 경우에만, Google 로그인을 통해 youtube.force-ssl 스코프의 접근 토큰을 발급받습니다.
  • 이용(Use): 발급된 토큰은 오직 사용자 본인 계정에 재생목록을 생성하고 곡을 추가하는 용도로만 사용합니다. 채널 정보 열람, 구독 관리, 기존 영상·재생목록 수정 등 다른 어떤 동작도 수행하지 않습니다.
  • 저장(Store): 서비스 서버는 어떠한 YouTube 사용자 데이터도 저장하지 않습니다. 접근 토큰은 사용자의 브라우저 메모리에만 존재하며, 페이지를 새로고침하거나 닫으면 즉시 소멸합니다. 서버로 전송되거나 디스크에 기록되지 않습니다(리프레시 토큰도 사용하지 않습니다).
  • 공유(Share): YouTube 사용자 데이터를 제3자에게 판매·양도·공유하지 않습니다. API 호출은 사용자의 브라우저가 Google 서버로 직접 수행하며, 서비스 서버를 경유하지 않습니다.

4-2. 데이터 삭제 정책

  • 서비스 서버에 저장되는 YouTube 사용자 데이터가 애초에 존재하지 않으므로, 별도의 서버 측 삭제 요청 절차가 필요하지 않습니다. 접근 토큰은 세션 종료(새로고침·탭 닫기) 시 자동 소멸합니다.
  • 브라우저에 남는 데이터는 localStorage의 플레이리스트 목록뿐이며(2항 참조), 좌측 메뉴에서 개별 삭제하거나 브라우저 데이터를 지우면 즉시 완전히 제거됩니다.
  • 이미 생성된 YouTube 재생목록은 사용자 본인의 YouTube 계정에 귀속되므로, YouTube 재생목록 관리 페이지에서 사용자가 직접 삭제할 수 있습니다.
  • 권한 철회: 언제든지 Google 계정 보안 설정 — 타사 앱 및 서비스 액세스 권한(https://myaccount.google.com/permissions)에서 이 앱의 접근 권한을 철회할 수 있습니다. 철회 즉시 본 서비스는 사용자의 YouTube 데이터에 접근할 수 없습니다.
  • 추가 문의나 삭제 요청은 sbb4113@gmail.com으로 연락해 주세요.

5. 이용하는 제3자 서비스

  • Groq — 요청 문장의 무드·에너지 해석(LLM API)
  • Google(YouTube Data API / OAuth) — "내 재생목록에 넣기" 기능(선택 사용)
  • Render — 백엔드(API 서버) 호스팅

  • umami — 방문자 통계. 쿠키를 사용하지 않고 개인을 식별하지 않는 방식으로 집계됩니다(예: 페이지뷰 수, 플레이리스트 생성 횟수 등 이벤트 카운트).

Privacy Policy (English summary)

This service is a non-commercial fan project. No account system or server-side database is used — your prompt text is sent to Groq (LLM provider) only to interpret your request and is not stored on our server. Saved playlists live only in your browser's localStorage.

The "Save to my playlist" feature uses YouTube API Services. By using it you agree to the YouTube Terms of Service, and the Google Privacy Policy applies.

The feature uses Google Sign-In with the youtube.force-ssl scope solely to create an unlisted playlist and add songs to your own YouTube account. It never reads channel info, manages subscriptions, or modifies existing videos.

How your data is handled. Access: an OAuth access token is obtained only when you explicitly click the button. Use: solely to create the playlist and insert the songs. Store: we store no YouTube user data on any server — the token lives in your browser's memory only, is never transmitted to or persisted by our backend (no refresh tokens are used), and is destroyed on refresh/close. Share: we never sell, transfer, or share YouTube user data with third parties; API calls go directly from your browser to Google.

Data deletion. Because no YouTube user data is stored on our servers, there is nothing to delete server-side; the token is discarded when your session ends. Playlists saved in your browser's localStorage can be removed from the side menu or by clearing browser data. Playlists already created belong to your own YouTube account and can be deleted at youtube.com/feed/playlists. You can revoke this app's access anytime at Google Account — Third-party apps & services (https://myaccount.google.com/permissions). Contact: sbb4113@gmail.com.
